Contador Angular

Este projeto foi desenvolvido com Angular e tem como objetivo demonstrar um contador interativo com manipulação de eventos e interface moderna.

Funcionalidades
➕ Aumentar o contador
➖ Diminuir o contador
🔄 Resetar o valor
🛠️ Tecnologias utilizadas
Angular (CLI 21.2.3)
TypeScript
HTML5
CSS3
▶️ Como executar o projeto
1. Clonar o repositório
git clone https://github.com/LeviRamoss/contador-angular.git
2. Acessar a pasta
cd contador-angular
3. Instalar dependências
npm install
4. Rodar o servidor
ng serve
5. Abrir no navegador
http://localhost:4200/
📂 Estrutura do projeto
src/
 ├── app/
